# 🎨 CommitPulse Themes

All 13 available themes for your CommitPulse badge. Use the `?theme=<slug>` query parameter to apply a
theme.

```
https://commitpulse.vercel.app/api/badge?username=YOUR_USERNAME&theme=<slug>
```
